# 🔒 Política de Seguridad

## Reportar una Vulnerabilidad

Si descubres una vulnerabilidad de seguridad en DownloaderMD, **no la reportes públicamente**.

### Cómo Reportar

1. **NO** abras un Issue público
2. Contacta directamente a través de [GitHub Security Advisory](https://github.com/Doriogamer/DownloaderMD/security/advisories)
3. O envía un email detallando:
   - Descripción de la vulnerabilidad
   - Pasos para reproducirla
   - Impacto potencial
   - Solución sugerida (si tienes)

### Tiempo de Respuesta

- **Confirmación**: Dentro de 48 horas
- **Actualización**: Dentro de 7 días
- **Parche**: Dentro de 14 días (en casos críticos)

## Prácticas de Seguridad

DownloaderMD implementa:

- ✅ Validación de URLs
- ✅ Sanitización de entrada de usuario
- ✅ No almacena contraseñas o datos sensibles
- ✅ Usa librerías actualizadas (yt-dlp, requests)
- ✅ Revisa dependencias regularmente

## Recomendaciones para Usuarios

- 🔄 Mantén Python actualizado
- 🔄 Actualiza DownloaderMD regularmente
- 🔄 Descarga solo de URLs que confíes
- ⚠️ No compartas archivos descargados de fuentes desconocidas
